1.3 Adding CSS Style and HTML Structure
CSS is code written to control the "style" of HTML elements.
CSS styles allow you to avoid writing the same code over and over.
CSS stands for Cascading Style Sheets. The C is the most important part of the phrase.
Most specific rules are applied to not just to a specified element, but also all of its descendants. This idea is called inheritance.
Properties that can be inherited are color, font, letter-spacing, line-height, list-style, text-align, text-indent, text-transform, visibility, white-space and word-spacing. Properties that cannot be inherited are background, border, display, float and clear, height and width, margin, min- and max-height and -width, outline, overflow, padding, position, text-decoration, vertical-align and z-index
More info on CSS
Inheritance prevents repetition when writing CSS

Box Sizing


1. HTML elements are boxes and each box has 4 components.
2. Because there are so many components to each box, it can often be hard to get the size of a box just right.
3. There are two techniques you can use to help deal with sizing issues.
1. Set sizes in terms of percentages rather than pixels.
2. Set the box-sizing attribute to border-box for every element.
4. Different browsers work slightly differently. Sometimes this causes different browsers to display the same code differently.

Box Positioning


1. Divs are block elements (as opposed to inline), so by default they take up the entire width of a page.
2. Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other.
